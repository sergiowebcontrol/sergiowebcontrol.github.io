Planteamiento del Problema

¿Cómo se puede clasificar solo un grupo imágenes que contengan elementos del lenguaje español, de un grupo de imágenes en general de manera automática y rápida?

Objetivo General

Desarrollar un modelo de aprendizaje automático (Machine Learning) que permita clasificar imágenes por su contenido, en este caso elementos del alfabeto español de una manera rápida y automática.

Objetivos Específicos

  • Interfaz gráfica de usuario
  • Set de datos confiable (Kaggle)
  • Entrenamiento en anaconda cloud
  • Entrenamiento en google colab
  • Modelo almacenado

Hipotesis

Utilizando redes neuronales convolucionales para la clasificación de imágenes, se logra realizar la interpretación de imágenes de letras del alfabeto español, de manera automática y rápida, con una confiabilidad del 90%.
